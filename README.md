memcache
===========

Very simple implementation of a memory cache.

Use
--------

For start:

    $ ruby memcache/lib/memcache.rb `<port>`

Default port is 5544.

Usage
-----

Just telnet to localhost `<port>`

Commands
--------

    set <key> <value>\r\n
    get <key>\r\n
    remove <key>\r\n
    list\r\n
    size\r\n
    quit\r\n

Enjoy!
